# 建立您自己的清單

- 閱讀 [Awesome 宣言](awesome.md) 與[清單指南](pull_request_template.md)，並確保您的清單符合規範。
- 在建立新清單之前搜尋此清單，因為您的清單可能是重複的。如果是，請嘗試為最好的那個做貢獻，而不是建立自己的。
- 您可能會發現[這個 Yeoman 生成器](https://github.com/dar5hak/generator-awesome-list)很有用。
- **在建立清單後至少等待 30 天再提交，以給予它成熟的機會。**
- **在提交將您的清單添加到此處的 Pull Request 之前，請務必再次閱讀[清單指南](pull_request_template.md)。**

感謝您的參與！ 😎
